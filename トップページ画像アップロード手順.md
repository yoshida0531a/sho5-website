# トップページ画像アップロード手順

## 現在の状況
✅ 5枚の写真を `/Users/akira/Pictures/shogo写真データ/top` に保存済み

## 次のステップ

### ステップ1: 画像をリポジトリにコピー

まず、保存した5枚の写真をこのリポジトリのソースディレクトリにコピーします：

```bash
# リポジトリのルートディレクトリに移動
cd /path/to/sho5-website

# ソースディレクトリを作成（まだ存在しない場合）
mkdir -p assets/images/top_sources

# 画像をコピー
cp /Users/akira/Pictures/shogo写真データ/top/*.{jpg,JPG,jpeg,JPEG,png,PNG} assets/images/top_sources/
```

または、Finderで直接コピー：
1. Finder で `/Users/akira/Pictures/shogo写真データ/top` を開く
2. 5枚の写真を選択してコピー（⌘+C）
3. このリポジトリの `assets/images/top_sources/` フォルダを開く
   （フォルダが存在しない場合は作成してください）
4. ペースト（⌘+V）

**注意**: `top_sources/` ディレクトリは `.gitignore` に含まれているため、
元画像はGitリポジトリにコミットされません。これにより大きな画像ファイルで
リポジトリが肥大化するのを防ぎます。

### ステップ2: 画像生成スクリプトを実行

リポジトリのルートディレクトリで以下のコマンドを実行：

```bash
cd scripts
./create-top-images.sh
```

または、npmコマンドでも実行できます：

```bash
npm run create-top-images
```

**このスクリプトが自動的に行うこと：**
- ソースディレクトリから最初の5枚を選択
- PC向け画像を生成（2000px幅）: `top_pc1.JPG` ～ `top_pc5.JPG`
- モバイル向け画像を生成（1200px幅）: `top_mobile1.JPG` ～ `top_mobile5.JPG`
- すべての画像を `assets/images/` に配置

**実行時の確認：**
- スクリプトは既存のトップ画像を上書きするか確認します
- `y` を入力して続行してください

### ステップ3: 結果を確認

生成された画像を確認：

```bash
ls -lh assets/images/top_pc*.JPG
ls -lh assets/images/top_mobile*.JPG
```

ブラウザで確認：
```bash
# ローカルサーバーを起動
python3 -m http.server 8000

# ブラウザで http://localhost:8000 を開く
```

### ステップ4: （オプション）画像を最適化

画像サイズをさらに削減したい場合：

```bash
cd scripts
./optimize-top-images.sh
```

または、npmコマンドでも実行できます：

```bash
npm run optimize-top-images
```

このスクリプトは：
- 画像を2000px幅に最適化
- ファイルサイズを約90%削減
- 元画像を `assets/images/backup_originals/` にバックアップ

### ステップ5: Gitにコミットしてプッシュ

変更をリポジトリに反映：

```bash
git add assets/images/top_pc*.JPG assets/images/top_mobile*.JPG
git commit -m "トップページ画像を更新"
git push
```

## トラブルシューティング

### `create-top-images.sh` がエラーになる場合

**エラー**: "ソースディレクトリが見つかりません"
- `assets/images/top_sources/` ディレクトリが存在するか確認
- このディレクトリに画像がコピーされているか確認

**エラー**: "ソース画像が不足しています"
- 最低5枚の画像（JPG/PNG形式）が必要です
- `ls assets/images/top_sources/` で画像を確認

**エラー**: "node コマンドが見つかりません"
- Node.js をインストール: https://nodejs.org/

### 画像の並び順を変更したい場合

画像は名前順に処理されるため、ファイル名を変更することで順序を調整できます：

```bash
cd assets/images/top_sources
mv photo1.jpg 01_photo1.jpg
mv photo2.jpg 02_photo2.jpg
# ...
```

## まとめ

**必須の手順：**
1. 画像を `assets/images/top_sources/` にコピー
2. `./scripts/create-top-images.sh` を実行
3. 結果を確認

**所要時間**: 約5分
